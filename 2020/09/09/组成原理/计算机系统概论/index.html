计算机组成原理之机器

第一章：系统概论

计算机系统的简介
一、计算机的软硬件概念
  • 现代计算机由软件和硬件两部分组成，软件分为系统软件和应用软件
  • 常见的系统软件有：语言处理程序，操作系统，服务性程序，数据库管理系统，网络软件
    二、计算机系统的层次结构
  • 系统复杂性的管理办法：抽象：高级的模型和低级的实体相对应，隐藏系统中不重要的细节
  • 物理结构的抽象：physics -> Devices -> Analog Circuits -> DIgital Circuits -> Logic -> Micro-architecture -> Architecture -> OS -> Application Software
  • 程序的抽象：微指令系统 -> 机器语言 -> 操作系统 -> 汇编语言 -> 高级语言
    tips:
计算机组成原理和计算机体系结构研究内容上的区别：
计算机系统结构：定义计算机系统软硬件的交界面，定义功能的实现方法，提供上层软件编写时和硬件交互的接口，指的是程序员所见到的计算机系统的属性，概念性的结构与功能特性（指令系统，数据类型，寻址技术，IO原理等）
计算机组成原理：实现计算机体系结构偶所体现的属性（具体指令的逻辑实现）
    计算机的基本组成
    一、冯.诺伊曼计算机的特点
  • 计算机由五大部件组成
    1）运算器：算术运算和逻辑运算
    2）存储器：存放数据和程序
    3）控制器：指挥程序的运行
    4）输入输出：信息转换
  • 指令和数据以同等地位存储在存储器，可按地址寻访
  • 指令和数据用而二进制表示
  • 指令由操作码和地址码组成
  • 程序存储
  • 以运算器为中心
    二、冯.诺依曼架构的改进
  • 以存储器为核心
  • 现代计算机的硬件框图：
    运算器 ：ALU 控制器 ：CU）—> CPU 存储器：主存和辅存） —-> 主机
    输入设备 输出设备） ——> I/O设备
    tips:
系统复杂性·管理的方法（3 y）
层次化（Hierachy）:将被设计的系统划分为多个模块或子模块
模块化（Modularity）:有明确定义（well-defined）的功能和接口
规则性（Regularity）:模块更容易被重用
三、计算机工作的步骤
    1. 上机前的准备：
      • 建立数学模型
      • 确定计算方法
      • 编制解题程序
        • 程序：运算的全部步骤
        • 指令：每一个步骤 格式：操作码+地址码
    1. 计算机的解题过程
      1）存储器的基本组成
      • 存储体-存储单元-存储元件
      • 存储单元：存放一串二进制代码
      • 存储字：存储单元中二进制代码的组合
      • 存储字长：存储单元中二进制代码的位数，每个存储单元赋予一个地址
      • 按地址寻址
      • MAR：存储器地址寄存器，反映存储单元的个数
      • MDR：存储器数据寄存器，反映存储字长

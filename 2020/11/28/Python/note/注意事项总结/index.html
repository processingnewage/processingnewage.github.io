注意事项总结

  1. 注意在可变类型之间的赋值问题
    # 变量相互之间的赋值要考虑变量的类型以及需要对其进行的操作
# 不可变类型变量相互之间的赋值：指向同一块内存，但由于不可变类型变量的不可修改性，故没有特别需要注意的地方
# 可变类型变量相互之间赋值：指向同一块内存，修改被赋值的变量会改变原变量的值，要保持新旧变量一致的情况下可忽略，但要只改变新的变量，需要使用到深浅拷贝
# example：
1）不可变类型
x = (1, 3)
y  = x
y.append(1)
print(y)

2）可变类型
x = [1, 3]
y = x

3）深浅拷贝
# 深拷贝
import copy
x = [1, 3]
y = copy.deepcopy(x)
y.append(4)
print(x, y)

4）深浅拷贝的区别
Python 直接赋值、浅拷贝和深度拷贝解析
* 直接赋值：其实就是对象的引用（别名）
* 浅拷贝(copy)：拷贝父对象，不会拷贝对象的内部的子对象
* 深拷贝(deepcopy)： copy 模块的 deepcopy 方法，完全拷贝了父对象及其子对象（深拷贝需要引入copy模块）
# example：
import copy
a = [1, 2, 3, 4, ['a', 'b']] # 原始对象
b = a                       # 赋值，传对象的引用
c = copy.copy(a)            # 对象拷贝，浅拷贝
d = copy.deepcopy(a)        # 对象拷贝，深拷贝
a.append(5)                 # 修改对象a
a[4].append('c')            # 修改对象a中的['a', 'b']数组对象
print( 'a = ', a )
print( 'b = ', b )
print( 'c = ', c )
print( 'd = ', d )

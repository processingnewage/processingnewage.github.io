变量与数据类型

  • 变量：内存空间地址的别名

  • 变量命名的基本原则：

    • 必须以字母、下划线开头。其他字符只能是字母、数字、下划线
    • 不能使用%、？之类的特殊符号，如空格、￥、&、#等
    • 大小写敏感
    • 不能使用关键字

  • 良好的编程习惯：

    • 使用描述性的名字，见名知意
    • 使用小写字母开头表示变量、全大写表示常量、大写字母开头表示自定义类型（class）
      ex:
      nick_name = “python命名法”
      nickName = “驼峰命名法”
      NickName = “在python中表示一个新的自定义类型”
      NICK_NAME = “全部字母大写，称为帕斯卡命名法，常用于命名常量”

  • python中没有常量机制！！！ 靠人为自觉执行 大写字母命名的量不去修改

  • 数字类型：整数类型 int 浮点型 float 复数 complex（comp_num = 4 +3j）

  • 标准数据类型：

    • 数字类型：Number
    • 字符串类型：String
    • 元组类型：Tuple
    • 列表类型：List
    • 集合类型：Set
    • 字典类型：Dictionary

  • input函数接受用户的输入，接受的类型一定是字符串类型

  • 强大的eval函数：python的解释器函数，可直接解析数据

  • 不同进制的整数类型：二进制、八进制、十进制、十六进制（BODH）

  • Python要求所有浮点数必须带有小数部分，用于区分浮点数和整数类型

  • 浮点数表示方法：

    • 十进制表示
    • 科学计数法：96e4、9.18e16、4.2e-3

  • 浮点数的小问题：浮点型15位小数内为有效小数，一旦超过16位，就会存在误差

  • 表示高精度小数：decimal模块

  • num = decimal.Decimal(‘3.1222222222313231221222222’) // 使用Decimal类型时必须使用单引号创建数字类型

  • 自定义浮点数字的精度：decimal.getcontext().prec = 120 // 120位小数

  • 引用的本质是变量存储地址

  • python采用基于值的内存管理方式，为不同的变量赋相同的值时，多个变量指向同一块内存地址

  • 数字类型的操作：

  • 基本算数运算符：由python解释器直接提供。不需要引用第三方库，也叫做内置操作符

  • 复合算数运算符

    • 操作符：
      • +、-、*、/、%
      • num1 // num2 地板除，求整商
      • num1 ** num2 num1的num2次幂

  • 在python，/除法表示和数学一致，除的结果一定为浮点数

  • 在python中 ++/– 没有特定含义，不表示+1/-1操作，仅表示正负

  • 特殊的格式化控制字符：\ 续行符 \r 回车回到行首 \b回退一格

  • 格式化输出字符串：

    • 使用字符串format方法 <模板字符串>.format(<逗号分隔的参数>)
    • 格式控制字符串左对齐，数字右对齐

  • 在python中求余和取模结果是不同的

  • 命名规则：

    • 大小写敏感：Java 是大小写敏感的，这就意味着标识符 Hello 与 hello 是不同的
    • 包名：全小写字母，以目录形式管理多层包结构
    • 类名：对于所有的类来说，类名的首字母应该大写。如果类名由若干单词组成，那么每个单词的首字母应该大写
    • 方法名/变量名：应该以小写字母开头。如果方法名含有若干单词，则后面的每个单词首字母大写
    • 源文件名：源文件名必须和类名相同（大驼峰）。当保存文件的时候，你应该使用类名作为文件名保存（切记 Java 是大小写敏感的），文件名的后缀为.java（如果文件名和类名不相同则会导致编译错误）

  • 修饰符分类：

    • 访问控制修饰符 : default, public , protected, private
    • 非访问控制修饰符 : final, abstract, static, synchronized

  • 变量初始化的含义：本质上是分配内存空间

    public class InitTest {
    	int[] arrrs;    // 成员变量初始化：对于成员变量，初始化不需要赋值或调用new()方法，默认初始化机制保证不会报错
    public static void main(String[] args) {
  		// 对于局部变量：需要显式初始化，否则会报"The local variable a may not have been initialized"错误
        // 引用数据类型初始化：分配内存空间，调用new()方法[动态初始化] 或 直接赋值[静态初始化]
        // 基本数据类型初始化：分配内存空间，赋具体的值
        // 总结：无论是什么数据类型，初始化的含义都是分配内存空间，只不过对于基本数据类型和引用数据类型来说，分配内存空间的方法和表现形式不同，具体见上一条
        int[] arrs = new int[2];
        int a = 1;
        String b = "hello world";
        System.out.println(arrs[0]);
        System.out.println(a);
        System.out.println(b);
    }
}

#Mechanism Design

Problem Statement

Consider a crowdsourcing platform that needs multiple computing devices to complete a microtask (e.g., labeling images). Each device has:

  • A private cost per task (depending on skill/effort)
  • An output quality

The platform must assign tasks and decide payments to ensure high-quality results.

:::info Task: Formulate this as a mechanism design problem.
(You do not need to solve the mechanism or prove properties, but only model it mathematically.) :::

Requirements

  1. Define the set of computing devices and their corresponding types
  2. Specify the decisions
  3. Indicate the utility function of each computing device (assuming quasi-linear)
  4. Formulate the objective function (social welfare or revenue maximization)
  5. Write the full optimization problem (including decision variables, objective, constraints)

Solution: Crowdsourcing Task Allocation & Payment Mechanism

Context: Under asymmetric information (device cost and quality are private), design allocation and payment rules satisfying incentive compatibility and individual rationality.

  • Platform: Minimize spending, maximize quality → maximize "cost-performance ratio" (Quality-Utility Maximization)
  • Devices: Maximize earnings

1. Device Set & Types

Let the set of computing devices be:

$$\mathbf{N} = {1, 2, \cdots, N}$$

Each device $i$ has type:

$$\theta_i = (c_i, q_i)$$

| Symbol | Meaning | |:------:|:--------| | $c_i$ | Cost per task | | $q_i$ | Quality |

2. Decisions

| Variable | Description | |:--------:|:------------| | $x_i \in {0,1}$ | Task assignment (1 = assigned, 0 = not assigned) | | $p_i$ | Payment to device $i$ |

Vector forms:

  • $\mathbf{x} = {x_1, x_2, \cdots, x_N}^T$
  • $\mathbf{p} = {p_1, p_2, \cdots, p_N}^T$

3. Utility Functions

Device $i$ (quasi-linear):

$$u_i = p_i - c_i x_i$$

Platform:

$$u_p = \sum_{i \in \mathbf{N}} q_i x_i - p_i$$

4. Objective Functions

Platform Revenue Maximization:

$$\max_{\mathbf{x},\mathbf{p}} ; u_p = \max_{\mathbf{x},\mathbf{p}} \sum_{i \in \mathbf{N}} q_i x_i - p_i$$

Social Welfare Maximization:

$$\max_{\mathbf{x},\mathbf{p}} (u_p + \sum_{i \in \mathbf{N}} u_i) = \max_{\mathbf{x},\mathbf{p}} \sum_{i \in \mathbf{N}} q_i x_i - c_i x_i$$

5. Full Optimization Problem

$$ \begin{aligned} \max_{\mathbf{x},\mathbf{p}} \quad & \sum_{i \in \mathbf{N}} q_i x_i - p_i \[6pt] \text{s.t.} \quad & \ \end{aligned} $$

| Constraint | Description | Formula | |:-----------:|:------------|:--------| | IC | Incentive Compatibility: truthful reporting of cost and quality | $u_i(c_i, q_i, c_{-i}, q_{-i}) \geq u_i(c_i', q_i', c_{-i}, q_{-i}), \forall i \in \mathbf{N}$ | | IR | Individual Rationality: non-negative participation utility | $p_i - c_i x_i \geq 0, \forall i \in \mathbf{N}$ | | Quantity | At least $k$ devices assigned tasks | $\sum_{i \in \mathbf{N}} x_i \geq k$ | | Binary | Task assignment is binary | $x_i \in {0,1}, \forall i \in \mathbf{N}$ |

Note: Relationship to forward auctions — Let declared cost and quality be $b_i$, and true cost and quality be $c_i$; this corresponds to bidding $b_i$ and true valuation $v_i$ in forward auctions.
